# Glow + FZF Content Browser - Usage Guide

Three new just recipes have been added to `/home/user/youtube/justfile` for browsing, previewing, and searching your presentation content.

---

## üìö Browse - Interactive Content Browser

**Command:**
```bash
just browse
```

**Features:**
- Lists all video directories with FZF fuzzy finder
- Live preview with glow-rendered markdown (70% split window)
- Automatically detects `presentation.md` or `slides.md`
- Sorted alphabetically for easy navigation

**Keyboard Shortcuts:**
- `Enter` - Open selected presentation in glow pager (full screen)
- `Ctrl-E` - Open presentation in your default editor ($EDITOR)
- `Ctrl-P` - Run the presentation with presenterm
- `Esc` - Exit browser

**Use Case:**
Perfect for quickly browsing through your entire video library and reading presentations in a beautiful markdown format.

**Command:**
```bash

```

**Example:**
```bash

just preview nvim-ai-coding
```

**Features:**
- Directly opens a specific folder's presentation in glow
- Handles both `presentation.md` and `slides.md`
- Error handling if no presentation file exists

**Use Case:**
When you know exactly which presentation you want to read, this provides instant access without the browser UI.

---

## üîç Search - Content-Aware Search

**Command:**
```bash
just search "<search-term>"
```

**Examples:**
```bash
just search "kubernetes"
just search "neovim"
just search "automation"
```

**Features:**
- Smart case-sensitive search (lowercase = case-insensitive, uppercase = case-sensitive)
- Searches across ALL `presentation.md` and `slides.md` files
- Shows line numbers and context
- Syntax-highlighted preview with glow
- Color-coded search results

**Keyboard Shortcuts:**
- `Enter` - Open full file in glow pager
- `Ctrl-E` - Jump to exact line in your editor ($EDITOR +line file)
- `Esc` - Exit search

**Use Case:**
Perfect for finding specific topics, code examples, or references across your entire presentation library.

These recipes require the following tools to be installed:

1. **fd** - Fast file finder
   ```bash
   # Install: apt install fd-find, brew install fd, cargo install fd-find
   ```

2. **fzf** - Fuzzy finder
   ```bash
   # Install: apt install fzf, brew install fzf
   ```

3. **glow** - Markdown renderer
   ```bash
   # Install: brew install glow, or download from https://github.com/charmbracelet/glow
   ```

4. **ripgrep (rg)** - Fast search (for search command)
   ```bash
   # Install: apt install ripgrep, brew install ripgrep
   ```

## üí° Tips

1. **Set your editor**: Ensure `$EDITOR` environment variable is set:
   ```bash
   export EDITOR=nvim  # or vim, code, etc.
   ```

2. **Browse workflow**: Use `just browse` as your main entry point, then:
   - Preview presentations with Enter
   - Edit with Ctrl-E if you need to make changes
   - Present with Ctrl-P when ready

3. **Search + Edit**: The search command is perfect for making quick edits:
   ```bash
   just search "TODO"  # Find all TODOs
   # Press Ctrl-E on any result to jump directly to that line
   ```

4. **Chaining commands**: Combine with other just recipes:
   ```bash
   just preview cv-pipeline  # Review content
   cd cv-pipeline && just present  # Then present
   ```
