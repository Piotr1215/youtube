# Cmdline Registers Demo

Commands to demo:
- `:%s/Ctrl-r Ctrl-w/new/g` - replace word under cursor
- `:%s/Ctrl-r //new/g` - replace last search pattern
- `Ctrl-r =system('date')` - insert command output
- `Ctrl-r =5*42` - insert calculation result
